순서
1. .git이 있는 폴더 생성한다. 
2. 그 폴더에 올릴 파일을 저장한다.
3. git remote add origin(보통 이이름 사용) (github repository 생성할 때 복사가능한 주소) 순으로 적어준다.
4. git add . 으로 스테이지로 올려주고
5. git commit -m "아무내용" 을 적어서 로컬로 올리고
6. git branch -M main //아마도 처음 생성할때만 브랜치 없어서 이거 만들어주고
7. git push -u origin main //이렇게 해주면 로그인창 뜨면서 로그인하면 github에 올라감

git remote add origin[원격저장소 주소]
git push -origin main 에서 stream 이라는 단어가 나온것은 origin과 local 사이의 흐름을 설정하지 않아서 에러가 나옵니다. git push -u -origin main 에서처럼 -u 옵션을 주세요
